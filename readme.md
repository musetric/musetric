# Musetric

Musetric is a vocal training application.

## Development

```bash
yarn
yarn dev
```

## Check

```bash
yarn
yarn check:security
yarn fix:deps
yarn check:ts
yarn fix:lint
yarn fix:translations
yarn fix:format
yarn test
```
